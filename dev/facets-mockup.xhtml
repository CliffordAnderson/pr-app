Faceted search mockup

Publisher: If no publisher is checked, all publishers are searched. If any publisher is checked, only checked publishers are searched.

Decade: If no decade is checked, all decades are searched. If any decade is checked, only checked deades are searched. Similarly, if a decade is checked but no year is checked within it, all years within it are searched. If any year within a decade is checked, only checked years within that decade are searched. Unchecking a decade closes it and unchecks all years within it; closing the decade by clicking the triangle remembers selected years. Opening a decade and checking any year within it automatically checks the decade.
